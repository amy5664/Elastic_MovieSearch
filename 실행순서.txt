## 1. DB(Elasticsearch) 실행 [공통/필수]
(프로젝트 최상위 폴더에서 실행)

1) 터미널 열기 (최상위 폴더 위치)
2) 실행 명령어:
   docker-compose up -d

3) 확인:
   인터넷 창에   접속 -> JSON 뜨면 성공

---

## 2. 플러그인 설치
DB가 켜진 상태에서 아래 명령어를 입력해야 한글 검색이 가능합니다

1) 플러그인 설치:
   docker exec -it elasticsearch bin/elasticsearch-plugin install analysis-nori

2) 재시작 (설치 적용):
   docker restart elasticsearch

---

## 3. 데이터 적재 (ETL)
DB에 영화 데이터를 채워 넣기

1) 폴더 이동:
   cd etl

2) 라이브러리 설치:
   pip install -r requirements.txt

3) 데이터 적재 실행:
   python Movie_El.py

4) 결과 확인:
   http://localhost:5601 (Kibana) 접속 -> 좌측 햄버거 메뉴 -> Dev Tools
   GET movies/_search  입력 후 실행 (데이터 보이면 성공)

---

## 4. 백엔드 서버 실행

1) IntelliJ 실행 -> 'Open' -> `backend` 폴더 선택
2) `src/main/resources/application.yml` 확인 (포트 8484 확인)
3) `BootMovieProjectApplication` 파일 실행 (Run)
   -> 콘솔에 "Started..." 뜨면 성공 (http://localhost:8484/api/search?q=사랑)

---

## 5. 프론트엔드 실행 [프론트 개발자]
사용자 화면을 켭니다.

1) 터미널에서 폴더 이동 (중요!):
   cd react

2) 라이브러리 설치 (최초 1회):
   npm install

3) 서버 실행:
   npm run dev

4) 접속:
   http://localhost:5173 (크롬 브라우저 권장)
